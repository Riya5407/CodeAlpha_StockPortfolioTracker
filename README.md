# 📈 Stock Portfolio Tracker

A simple Python-based Stock Portfolio Tracker that allows users to enter stock symbols and quantities, calculates the total investment value using predefined stock prices, and saves the portfolio summary to a text file.

## 🚀 Features

* Track multiple stocks in a portfolio
* Calculate investment value based on stock quantity and price
* Display a detailed portfolio summary
* Save portfolio data to a text file
* Handle invalid stock symbols gracefully

## 🛠️ Technologies Used

* Python 3
* Dictionaries
* Loops
* Conditional Statements
* File Handling

## ▶️ How to Run

2. Navigate to the project folder:

```bash

```

3. Run the program:

